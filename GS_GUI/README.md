# 🚀 Rocket Telemetry Desktop Console (v8)

A high-performance Tkinter-based telemetry dashboard for rocket data.

## Features
- Live Serial + Simulator
- Playback with scrubber
- Rocket body-axis attitude view
- Map with pan/zoom + full track
- CSV logging + buffer export
- 5-minute sliding charts (no slowdown)

## Install
pip install -r requirements.txt

## Run
python rocket_telemetry_tk_v8_5min_window.py
